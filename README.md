### Kurulum Adımları

1. Projeyi bilgisayarınıza indirin:

```bash

git  clone [https://github.com/Matttiuz/Kaya-Tatlicilari-main]

```

  

2. Proje klasörüne gidin:

```bash

cd [Kaya-Tatlicilari-main]

```

  

3. Gerekli paketleri yükleyin:

```bash

flutter  pub  get

```

  

4. Uygulamayı başlatın:

```bash

flutter  run

```
